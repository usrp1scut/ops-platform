This is the embedded fallback page for the React/Vite portal mounted at /portal/. The Go binary was compiled without a real web build copied into the embed directory.

To populate it locally:

cd web
MSYS_NO_PATHCONV=1 VITE_BASE=/portal/ npm run build
cp -R dist/. ../internal/httpserver/ui/v2/static/
go build ./cmd/ops-api

The Docker build does this automatically via the web-builder stage. The legacy classic-script console is still served at /portal-legacy/ while the migration is observed.
